## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/eng-hussein-saad/portfolio.git
   ```

2. Navigate into the project directory:
   ```bash
   cd portfolio
   ```

3. Open the `index.html` file in your preferred browser.
